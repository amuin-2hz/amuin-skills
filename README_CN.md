# Amuin · 不做牛马

> 你的电脑，应该伺候你。不是你伺候它。

**Amuin** = "Am I a machine?" —— 来自打工人的一句无声反问。

我们分享一些轻量、实用、免费的小工具。让你的电脑不卡、硬盘不爆、工作流不闹心。没有付费优化大师，没有看不懂的参数，没有"重装系统就好了"的无奈。装进 AI Agent，说句话就帮你搞定。

## 为什么叫"不做牛马"？

牛马是工具。人不是。

我们选择不做牛马——把自己的电脑伺候好，不被卡顿和广告绑架，不把时间和钱花在本来能免费解决的事情上。

C 盘红了、电脑卡了，商家只会说"换个大的 SSD"。备份一晚上吃掉 200G，客服告诉你"开个 iCloud 会员"。

这些明明有免费解法。但解法藏在论坛的十七楼回复里，藏在五年前的图文教程里，或者写教程的人默认你有程序员基础。

Amuin 把这些解法收拾干净，做成技能教给你的 AI Agent。Agent 学会了，就全自动帮你搞定——扫描、判断、安全迁移、验证结果。每个工具都在真实机器上跑过，每一步都带着安全检查。

帮你的电脑多撑两年，不就是帮你省钱了？

## 怎么用

把技能装到你的 AI Agent 里（比如 Claude Code），它就能按我们的脚本自动帮你搞定。

不用记命令，不用看教程。你只需要说一句"帮我把磁盘清一下"，Agent 自己会扫描、判断、安全迁移、验证结果。

每个工具里也带了手动跑的脚本，如果你喜欢自己动手。
